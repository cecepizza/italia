# Italian Property Family Dashboard

Build a web dashboard that:

1. Shows a map of Italy with our target towns marked
2. Displays current exchange rates EUR/USD
3. Lists newest properties in each town
4. Shows weather in each location
5. Tracks everyone's research assignments and progress
6. Includes a family voting system for properties
7. Displays countdown to scouting trip
8. Shows cost of living comparisons
9. Aggregates news from Italian property blogs
10. Includes a shared notes section

Make it mobile-friendly and update automatically.
Use free APIs where possible and create a simple deployment guide.
