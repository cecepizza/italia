# Comprehensive Italy Research Automation

Create scripts that automatically:

1. PROPERTY MONITORING:
   - Check Immobiliare.it, Casa.it, and Idealista daily
   - Screenshot interesting properties
   - Alert on new listings under our budget
   - Track property price history

2. LOCATION RESEARCH:
   - Compile crime statistics for each town
   - Find nearest hospitals and their ratings
   - Check internet speeds and provider options
   - List local schools and amenities
   - Find expat groups and forums

3. TRAVEL PLANNING:
   - Monitor flight prices to Bari, Catania, Rome
   - Find Airbnbs in target areas
   - Create optimal route between towns
   - Compile local real estate agent contacts

4. DOCUMENTATION:
   - Generate weekly PDF reports
   - Update shared Google Sheets
   - Create property comparison matrices
   - Build image galleries of each town

5. COMMUNICATION:
   - Send daily updates to family WhatsApp/Slack
   - Create Instagram-style property cards
   - Generate "Property of the Week" emails

Output everything to organized folders and Google Drive.
